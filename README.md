# Data Organisation

This repository contains assignments and practical exercises for the **Data Organisation** course. It focuses on fundamental concepts related to how data is structured, stored, and efficiently managed in computer systems.

## Contents

* Algorithm applications
* Practical exercises and examples
* Course assignments

## Purpose

The purpose of this repository is to support learning and experimentation with data organisation principles such as data structures, memory representation, and efficient data management techniques.

## Usage

You can review the assignments by exploring the folders.
